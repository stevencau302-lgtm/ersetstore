🚚 API Ongkir (api.co.id)

Konfigurasi cek ongkir lewat api.co.id. Dapatkan API key di dashboard.api.co.id. Pengaturan disimpan di browser ini (localStorage).

Kode desa toko kamu. Cari di cascade wilayah saat checkout, atau di API Regional.

🔒 Catatan Keamanan

API key disimpan di localStorage browser, jadi hanya tersimpan di perangkat ini dan tetap terlihat di sisi klien saat request dikirim. Untuk situs produksi yang aman, sebaiknya simpan API key di backend/server dan teruskan request ongkir lewat proxy, supaya key tidak bocor ke pengunjung.
